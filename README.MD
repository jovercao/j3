# j3

## core

整个j3的核心模块，包括以下内容：

- [module](./core/module/README.MD) - 模块化核心库
- [db](./core/db/README.MD) - 数据库基础库
- [svc](./core/svc/README.MD) - 远程服务
- [util](./core/util/README.MD) - 工具库
